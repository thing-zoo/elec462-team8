코드 실행방법!

1. server 파일을 원하는 디렉토리에 넣고 ./server를 통해 실행한다.
(그러할 경우 포트는 13000 고정, 자신의 ip 주소로 서버가 만들어집니다.)

2. client 파일을 같은 네트워크 내의 컴퓨터의 원하는 디렉토리에 저장하고 ./client 192.168.43.123 (서버의 아이피 주소) 로 실행한다.
(아이피주소는 ifconfig 명령어를 치면 알 수 있어요!)

3. client 의 UI에 따라 아이디를 만들고 로그인을 하면, 서버 컴퓨터의 디렉토리에 자신의 아이디로 된 폴더가 생성되고,

4. 파일을 서버에 업로드 하거나, 다운로드 받을수도 있고, 서버에  있는 자신의 폴더의 내용을 볼 수도 있으며, 삭제도 가능합니다.

5. 로그아웃을 하면 사용 종료가 됩니다.




최신 버젼으로 업데이트를 원하는 경우!

1. 저희 깃허브 주소에 접속합니다. ( https://github.com/gon125/KSdrive.git )

2. clone을 하신 후, 터미널 창에 make server 와 make client를 실행합니다.

3. 새로 업데이트 된 server 와 client 파일을 코드실행방법! 대로 사용하시면 됩니다.


저희 서비스를 이용하면, linux 환경에서 학생들의 과제 제출 원격 저장소로도 활용이 가능합니다. 사용자의 컴퓨터의 server 파일이 있는 디렉토리에
학번을 아이디로하여 웹에 업로드하고 다운로드 받는 과정 없이 채점하기가 용이합니다.
